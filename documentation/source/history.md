### 2025-xx-xx (SIL WSTech team) BJCree version 7.000
- Changed to use Open Font License, sources available on GitHub
- Changed to UFO-based workflow
- Replaced Century Schoolbook Latin glyphs with new design based loosely on C059 (New Century Schoolbook) from (URW)++, released under the SIL Open Font License version 1.1.
- Added new characters:
    U+142E ᐮ an-cans
    U+142D ᐭ on-cans
